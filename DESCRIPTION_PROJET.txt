───────────────────────────────────────────────────────────────────────────────
1. PRÉSENTATION GÉNÉRALE
───────────────────────────────────────────────────────────────────────────────

Application web full-stack moderne pour la gestion complète d'une entreprise
laitière. Couvre tous les aspects opérationnels: production, stocks, clients,
commandes, livraisons, facturation, et analyses.

OBJECTIFS:
• Centraliser toutes les opérations en une plateforme unique
• Automatiser les processus récurrents et notifications
• Optimiser gestion des stocks et production
• Améliorer suivi des commandes et livraisons
• Faciliter facturation et suivi financier
• Fournir analyses détaillées pour décisions stratégiques

───────────────────────────────────────────────────────────────────────────────
2. ARCHITECTURE TECHNIQUE
───────────────────────────────────────────────────────────────────────────────

BACKEND (API REST):
├── Runtime: Node.js 18+
├── Framework: Express.js
├── Langage: TypeScript
├── Base de données: MySQL 8.0 + Sequelize ORM
├── Authentification: JWT (JSON Web Tokens)
├── Sécurité: bcryptjs, Helmet, CORS, Rate Limiting
├── Validation: Express-validator
├── Logging: Morgan, Winston
├── Emails: Nodemailer
└── Automatisation: Node-cron

FRONTEND (Interface):
├── Framework: React 18 + TypeScript
├── Build: Vite (ultra-rapide)
├── Routage: React Router v6
├── Styling: Tailwind CSS (responsive)
├── HTTP: Axios
└── État: Context API

BASE DE DONNÉES (6 tables principales):
├── Users (utilisateurs/authentification)
├── Products (catalogue produits laitiers)
├── Batches (lots de production/traçabilité)
├── Clients (informations clients)
├── Orders (commandes et livraisons)
└── Invoices (facturation et paiements)
